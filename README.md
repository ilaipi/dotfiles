# For win10 WSL

## Windows Terminal
把 `Windows Terminal config.json` 内容复制到`Windows Terminal` 的配置文件中

## 安装软件
[安装xcxsrv](https://sourceforge.net/projects/vcxsrv/)，然后双击`config.xlanch`
[安装auto hot key](https://www.autohotkey.com/)
